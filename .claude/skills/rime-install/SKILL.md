## 步骤

1. 去 https://rime.im/download/ 寻找用户对应的操作系统的最新下载地址
  - 可能还得进入上面网页中提供的github release latest页面看看（如果有），版本是否是新的（如果有差距，向用户说明）
  - 额外信息
    - windows可以通过 winget 安装 Rime.Weasel

2. 确认用户是否想要使用包管理工具。优先推荐用户使用包管理器（如果能用）

3. 如果是以下载文件的方式，向用户确认开始下载的文件的地址、是否开始下载

4. 开始安装前，向用户确认是否进行安装
